A premium web-based player for [Radio Paradise](https://radioparadise.com/), built with React, TypeScript, and Vite.

**Tuner** offers a dedicated, high-quality listening experience for all four Radio Paradise mixes, complete with real-time metadata, album art, and lossless audio support.

## Features

-   **All 4 Mixes**: Access Main, Mellow, Rock, and Global mixes.
-   **Audio Quality Options**: Support for FLAC (Lossless), AAC (High Quality), and MP3 (Compatible).
-   **Real-time Metadata**: Displays current artist, song, album, and cover art.
-   **Modern UI**: Clean, responsive interface designed for music lovers.

## Tech Stack

-   **Framework**: [React](https://react.dev/)
-   **Language**: [TypeScript](https://www.typescriptlang.org/)
-   **Build Tool**: [Vite](https://vitejs.dev/)
-   **Styling**: CSS (Custom Design)

## Getting Started

### Prerequisites

-   Node.js (v18 or higher recommended)
-   npm

### Installation

1.  Clone the repository:
    ```bash
    git clone https://github.com/yourusername/tuner.git
    cd tuner
    ```

2.  Install dependencies:
    ```bash
    npm install
    ```

3.  Start the development server:
    ```bash
    npm run dev
    ```

## Development

The project structure is organized as follows:

-   `src/`: Source code including components and logic.
-   `public/`: Static assets.
